# Interfaz para un CRUD

Basándome en <a href="https://startbootstrap.com/theme/sb-admin-2" target="_blank">esta</a> plantilla de StartBootstrap, he creado la interfaz para la aplicación que debemos entregar a final de curso.

La parte CRUD de la tabla está hecha con los snippets de <a href="https://datatables.net" target="_blank">DataTables</a> en base al plugin utilizado en la plantilla anterior.

Se trata de una aplicación web para la gestión de una clínica.
En esta primera versión, sólo tendrán opción de acceso administradores y empleados. En actualizaciones futuras se pretende que los pacientes también tengan acceso a su historial médico, puedan hacer algunas modificaciones en su perfil, solicitar cita e incluso comunicarse con sus doctores mediante la app.
